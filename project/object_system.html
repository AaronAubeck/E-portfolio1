Object System
Doctors Surgery

Throughout the object oriented systems module we were taught about how to develop and plan an object system. as a final project for this module we had to create a doctors surgery system using the skills we learnt. This is my system.

A key focus when working on this project was the implementation of classes and functions throughout those classes. For the system to work correctly it required an understanding of the relationships between these classes and how to implement them based off of a UML diagram.
